International Groundwater Resources Assessment Centre (IGRAC)

Delft, Netherlands

May 2023 - October 2023

At IGRAC, I spearheaded the development of a groundbreaking methodology for reporting on the quantitative status of groundwater bodies worldwide. This work was a crucial contribution to the World Meteorological Organization's (WMO) State of Global Water Resources report, establishing new standards for global groundwater assessment.

Key Achievements:

  • Developed pioneering Python script for evaluating aquifer health, comparing 17 methodologies through sophisticated time-series analysis
  • Created tools utilized by World Meteorological Organization (WMO) for global water reports
  • Analyzed over 1 trillion data points using AI and Python algorithms
  • Processed data from 4,084 groundwater monitoring wells across 8 states

Technical Innovations:

  • Implemented season-adjusted quarterly resampling methods for enhanced trend analysis
  • Developed comprehensive data filtering pipeline using advanced statistical methods
  • Applied Linear Regression and Mann-Kendall tests at 95% confidence level
  • Created sophisticated interpolation maps and visualizations for aquifer health assessment

This work significantly advanced IGRAC's capabilities in global groundwater monitoring, providing a robust framework for assessing groundwater trends worldwide. The methodology has been adopted by WMO, demonstrating its value for international water resource management.
